## 2단계 기능 구현 목록
- [ ] 노선별 추가 요금
  - [ ] 여러 노선을 환승할 경우 가장 높은 추가 요금을 적용
- [ ] 연령별 요금 할인
  - [ ] 13 ~ 18세는 청소년 요금 할인
    - [ ] 요금에서 350원을 제하고 20% 할인
  - [ ] 6 ~ 12세는 어린이 요금 할인
    - [ ] 요금에서 350원을 제하고 50% 할인
  - [ ] 6세 미만은 무료
  - [ ] 65세 이상은 무료
